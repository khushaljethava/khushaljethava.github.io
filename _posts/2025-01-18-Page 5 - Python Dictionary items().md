## items() Parameters 

The items() method does not take any parameters as arguments.

Let see an example of the items() method in python.

### Example 1: How to use items() method in python dictionaries?

```python
cars = { 1 : "BMW", 2 : "TOYOTA", 3 : "TATA"}

print(cars.items())

```
